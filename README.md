#复利计算器
###使用语言：*c*
###使用工具：*VC++*
</br>
* 制作了一个简单的进入主界面，增强可视性。
* 使用c语言用简单的*if*语句和*switch*等语句，实现选择功能，从而进行计算。
* 数学计算方面目前只用到了*pow（x,y)*幂函数。
